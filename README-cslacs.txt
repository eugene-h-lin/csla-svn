This is CSLA .NET version 3.6, for .NET 2.0 Framework with SP1

Version 3.6.0 of the framework is documented in the
Expert C# 2008 Business Objects book from Apress.

This download may contain a later version of the framework, and so
the code may not exactly match that in the book. For information 
on installation, book errata, updates to the framework 
and so forth, please visit http://www.lhotka.net/cslanet

NOTE: The ProjectTracker reference app has been moved to the
Samples download for CSLA .NET for Windows.


This zip file contains the following folders:

cslacs\
The CSLA .NET framework code.

dependencies\LinqBridge
The open source reimplementation of LINQ 2 objects for .Net 2.0
